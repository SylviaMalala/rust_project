# Quick Reference Guide

## Running the Examples

### 1. Main Task Manager
```bash
cargo run
```

### 2. Hello World
```bash
cargo run --example hello_world
```

### 3. User Input Demo
```bash
cargo run --example user_input
```

### 4. Data Types Demo
```bash
cargo run --example data_types
```

## Quick Commands

### Build project
```bash
cargo build
```

### Check for errors (fast)
```bash
cargo check
```

### Format code
```bash
cargo fmt
```

### Run tests
```bash
cargo test
```

### Clean build artifacts
```bash
cargo clean
```

## Common Cargo Commands

| Command | Description |
|---------|-------------|
| `cargo new <name>` | Create new project |
| `cargo init` | Initialize in existing directory |
| `cargo build` | Compile project |
| `cargo run` | Compile and run |
| `cargo test` | Run tests |
| `cargo doc --open` | Generate and open documentation |
| `cargo clean` | Remove build artifacts |
| `cargo update` | Update dependencies |

## File Paths

- Main code: `src/main.rs`
- Examples: `examples/*.rs`
- Config: `Cargo.toml`
- Documentation: `TOOLKIT.md`
- AI Journal: `AI_PROMPT_JOURNAL.md`
